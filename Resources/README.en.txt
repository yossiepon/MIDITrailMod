(1) Introduction

  MIDITrail is a MIDI player which provides 3D visualization of
  MIDI datasets. MIDITrail supports SMF format 0/1, and multiple
  MIDI ports.

(2) System requirement

  OS: Windows 7 / 8 / 10
  VIDEO: Support for DirectX 9.0c graphics

(3) Quick Start

  1. Drop Standard MIDI file(*.mid) to the window.
  2. Push space key to play/pause, and push ESC key to stop.

(4) User manual

  Please check "Help" -> "Manual...".

(5) About DirectX End-User Runtime

  If you get "d3dx9_**.dll was not found" error, please install
  "DirectX End-User Runtime".

  DirectX End-User Runtimes (June 2010) 
  https://www.microsoft.com/en-us/download/details.aspx?id=8109
